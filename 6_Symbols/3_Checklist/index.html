IT Team Member Onboarding Checklist

0% Complete

Access and Accounts

  • Company Email Setup
    Create and configure company email account with appropriate groups and permissions
  • VPN Access
    Set up VPN credentials and provide configuration documentation
  • System Credentials
    Create accounts for essential systems (Active Directory, development environments, etc.)

Hardware and Software

  • Workstation Setup
    Prepare and configure workstation with standard IT image
  • Required Software Installation
    Install and configure all necessary software and development tools
  • Mobile Device Setup
    Configure company mobile device and/or MDM enrollment if applicable

Training and Documentation

  • Security Training
    Complete required security awareness and compliance training
  • Internal Systems Training
    Review documentation and complete training for internal systems and tools
  • Department Procedures
    Review IT department procedures, on-call schedules, and escalation protocols

Team Integration

  • Team Introduction
    Schedule team meet-and-greet and introduction to key stakeholders
  • Project Overview
    Review current projects and responsibilities with team lead
  • Mentor Assignment
    Assign mentor/buddy for first month of employment
